# module_names_lower_snake

## Script
- `plugins/module_names_lower_snake.ast.py`

## Description
- Modules must use lower_snake_case
- Consistent module naming improves readability and automation.
## Good

```systemverilog
module my_module;
endmodule
```

## Bad

```systemverilog
module MyModule;
endmodule

module MY_MODULE;
endmodule
```
